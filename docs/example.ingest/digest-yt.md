:movie_camera: Autoresearch, Agent Loops and the Future of Work:link: [Watch on YouTube](https://youtube.com/watch?v=nt9j1k2IhUY)

 :us: Andrej Karpathy released "Auto Research," a minimal system where an AI agent autonomously iterates on LLM training code in a loop — each run lasts 5 minutes, improvements are kept via git commits, and regressions are discarded, all guided by a human-written strategy document (program.md). The video argues this "agentic loop" pattern is a new work primitive extending far beyond ML research: any task with a measurable score, fast iterations, and bounded action space can be automated similarly. The community highlights connections to the "Ralph Wiggum" technique and the open question of scaling these loops into collaborative multi-agent swarms.

 :taiwan: Andrej Karpathy 發布了「Auto Research」專案，讓 AI 代理在迴圈中自主修改 LLM 訓練程式碼——每次實驗限時 5 分鐘，改善就保留，退步就捨棄，全程由策略文件引導。影片主張這種「代理迴圈」模式是全新的工作原語，不僅適用於 ML 研究，任何具備可量化指標與快速迭代的任務都能採用。社群也將其與「Ralph Wiggum」技術連結，並探討如何擴展為多代理協作群體。
